## ğŸ”¹ Project 1: Responsive Contact Form

### Description
A responsive contact form created using HTML and CSS with client-side validation implemented using JavaScript. The form adapts to different screen sizes and ensures that user inputs are validated before submission.

### Features
- Structured input fields for user details  
- JavaScript validation for required fields  
- Clean and responsive layout  
- Works on desktop and mobile devices  

### Technologies Used
- HTML5  
- CSS3 (Flexbox and responsive design)  
- JavaScript (Form validation and DOM access)  

### Learning Outcomes
- Building structured forms using HTML  
- Validating user input using JavaScript  
- Creating responsive layouts  
- Understanding form handling logic

## ğŸ”¹ Project 2: Dynamic To-Do List

### Description
A dynamic To-Do List application that allows users to manage daily tasks. The application uses browser localStorage to store tasks permanently without using any backend.

### Features
- Add new tasks  
- Mark tasks as completed  
- Delete tasks  
- Persistent data storage using localStorage  
- Responsive and user-friendly interface  

### Technologies Used
- HTML5  
- CSS3  
- JavaScript (DOM manipulation and events)  
- Browser localStorage  

### Learning Outcomes
- DOM manipulation using JavaScript  
- Event handling  
- Using localStorage for client-side data persistence  
- Dynamically updating the user interface
